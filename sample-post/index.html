Computing a limit using Taylor/Maclaurin series

April 14, 2021

Some limits lead to an indeterminate form, that is, a 0/0 expression. Let’s look for example at,

limx→0x−tan(x)x3

One way to solve this problem is to apply L’Hôpital’s rule, since we have a numerator (xtan(x)) and denominator (x3), both which go to zero as x→0, making it suitable for the rule. However, applying it only once doesn’t solve the problem, as

limx→0x−tan(x)x3=limx→0d(tan(x))dx−dxdxd(x3)dx=limx→01−sec2(x)3x2

Now we might be tempted to split the limit in two terms, but that will only complicate things because it will create a term 13x2→∞. Instead we will either consider that 1−sec2(x)→0 or substitute it by the equivalent −tan2(x)→0. Here we choose to work with the first option. Differentiating again leads to

limx→01−sec2(x)3x2=limx→0−2sec2(x)tan(x)6x

Which is again an indeterminate form. We notice that we keep having an indeterminate form because we cannot cancel a power term xn with a trigonometric function (unless we expand that function as a power series which we’ll see in this post), which is necessary to eliminate the indeterminate form, so we have to differentiate as many times as necessary to reduce the xn to a 1. A final step is thus

limx→0−2sec2(x)tan(x)6x=limx→0−2(2sec2(x)tan2(x)+sec4(x))6=−26=−13

However, there is an alternative way to compute this limit. It is possible to expand tan(x) as a Taylor series, or more specifically, the Maclaurin because we will be expanding around x=0. The first few terms of the series for any analytical function are

f(x)=f(0)+df(x)dx∣∣x=0x+d2f(x)dx2∣∣x=0x22!+d3f(x)dx3∣∣x=0x33!+⋯

The first three derivatives of tan(x) are

d(tan(x))dx=sec2(x) d(sec2(x))dx=2sec2(x)tan(x) d(2sec2(x)tan(x))dx=2(2sec2(x)tan2(x)+sec4(x))

At x=0, cos(x)=1 and sin(x)=0, so sec(x)=1 and tan(x)=0. Therefore, with f(x)=tan(x), f′(0)=1, f′′(0)=0 and f′′′(0)=2. The Taylor series for tan(x) is then

tan(x)=x+26x3+⋯

Plugging this into our original limit, we get

limx→0x−x−26x3x3=limx→0−26x3x3=−26=−13

And there it is. The limit simplifies to −13. The higher order terms need not be included because on the limit as x→0 they will vanish.
